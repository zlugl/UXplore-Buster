📖 Debugging & Optimization Guide

🔍 Common HTML Bugs

  • Unclosed Tags: <div><p>Text</div> (missing </p>)
  • Missing type attribute: <input placeholder="Email"> should be <input type="email">
  • Missing alt on images: <img src="pic.jpg"> should have alt="description"
  • Wrong nesting: Block elements inside inline elements

🎨 Common CSS Bugs

  • Fixed height overflow: height: 50px with tall content = hidden text
  • Low contrast: Light gray text on white = unreadable
  • Width issues: width: 100% + padding = box overflow
  • Alignment problems: Margins and padding causing misalignment
  • Missing responsive: Looks bad on mobile without @media queries

✅ Debugging Process

  1. Identify: What's broken? What should it be?
  2. Isolate: Which line of code caused it?
  3. Review: Check syntax, values, logic
  4. Fix: Make the correction
  5. Test: Verify the fix works
