📖 Lecture: Core Design Principles

UI Elements & Button Design

Buttons are critical UI elements. Users interact with buttons hundreds of times daily. Choosing the right button shape conveys meaning and improves usability.

Button Shapes & Their Meanings:

●●●

Rounded/Pill Buttons

Best for: Primary actions, calls-to-action, friendly interfaces
Feels: Soft, approachable, modern
Example: "Sign Up", "Subscribe", "Start Now"

▭▭▭

Square/Rectangle Buttons

Best for: Professional, serious applications
Feels: Professional, structured, traditional
Example: "Login", "Submit", "Confirm"

◆◆◆

Slightly Rounded Buttons

Best for: Balanced designs, general purposes
Feels: Modern, approachable but still professional
Example: Secondary actions, navigation

❤❤❤

Heart/Custom Shapes

Avoid for: Login, primary actions, critical functions
Better for: Favorites, likes, decorative purposes only
Issue: Confusing on login forms

Rule: For login forms, use clear rectangular or slightly rounded buttons. Users expect standard, recognizable buttons for authentication.

Typography & Font Selection

Typography is a crucial design decision. Font style dramatically affects readability, professionalism, and emotional response.

Font Styles:

Serif Fonts

Feeling: Traditional, professional, formal
Use for: Academic papers, books, formal applications
Readability: Great for print, challenging on screens

Sans-Serif Fonts

Feeling: Clean, modern, professional
Use for: Web design, UI, contemporary products
Readability: Excellent for screens and web

Cursive Fonts

Feeling: Artistic, elegant, flowing
Use for: Logos, headings, special occasions
Readability: Poor for body text and posters

🚫 Avoid for Posters

For posters: Use bold sans-serif or display fonts
Not: Cursive fonts (hard to read from distance)
Best practice: Capital letters in sans-serif for visibility

For a poster display (outdoor, etc.): Use BOLD CAPITAL SANS-SERIF fonts. They're readable from distance. Cursive is terrible for posters!

Color Theory Basics

Colors evoke emotions and guide user attention.

Red

Energy, urgency, danger (use for alerts)

Cyan/Blue

Trust, calm, professional

Green

Success, growth, safe

Yellow/Orange

Warning, attention, warmth

Visual Hierarchy

Visual hierarchy guides users' eyes to important content.

  • Size: Larger elements appear more important
  • Color: Bright colors stand out from neutral backgrounds
  • Contrast: High contrast draws attention
  • Position: Top-left and center are noticed first
  • White Space: Isolation makes elements prominent
