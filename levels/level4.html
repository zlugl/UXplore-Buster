📖 Lecture: HTML & CSS Fundamentals

What is HTML?

HTML (HyperText Markup Language) is the foundation of all web pages. It provides the structure and content of web pages using "tags" or "elements."

Think of HTML as the skeleton of a building - it defines what's there.

Basic HTML Elements:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
    <button>Click Me</button>
  </body>
</html>

<h1> to <h6>: Headings (h1 is largest)
<p>: Paragraphs of text
<button>: Interactive buttons
<input>: Form fields for user input
<div>: Generic container for sections
<img>: Images

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout HTML elements. It controls colors, fonts, spacing, borders, and positioning.

Think of CSS as the makeup and furnishings - it makes the building look good.

Basic CSS Syntax:

button {
  background-color: #00d9ff;
  color: #0a0e27;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  font-size: 16px;
  cursor: pointer;
}

background-color: Button background color
color: Text color
padding: Space inside the element
border: Border around element
border-radius: Rounded corners
font-size: Text size
cursor: Mouse cursor style

HTML and CSS Together

Here's a simple login form combining HTML and CSS:

<html>
<head>
  <style>
    .login-form {
      max-width: 300px;
      margin: 50px auto;
      padding: 20px;
      border: 1px solid #00d9ff;
      border-radius: 8px;
    }
    input { width: 100%; padding: 10px; margin: 10px 0; }
    button { width: 100%; padding: 10px; background-color: #00d9ff; }
  </style>
</head>
<body>
  <div class="login-form">
    <h2>Login</h2>
    <input type="text" placeholder="Username">
    <input type="password" placeholder="Password">
    <button>Sign In</button>
  </div>
</body>
</html>

Essential CSS Properties for UI Design

Layout

  • display: flex, grid
  • margin, padding
  • width, height
  • position: absolute, relative

Styling

  • background-color
  • color, font-size
  • border, border-radius
  • box-shadow, opacity
